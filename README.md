## Weather-App

This a 5 day weather forecasting app built with React, the app allows the user to search for the weather within the UK.

![Screenshot](images/screenshot.png)

## Install

```sh
npm install
```

## Usage

```sh
npm start
```

## Run tests

```sh
npm test
```
